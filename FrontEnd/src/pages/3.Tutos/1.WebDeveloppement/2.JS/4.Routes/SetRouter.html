Introduction

Bien Architecturer le code est essentielle pour plusieurs raisons : la maintenabilité, l'évolutivité ou la sécurité

Nous allons créer un dossier routes à la racine du projet.

A l’intérieur de ce dossier, nous créons un dossier api.

Dans le dossier api, nous créons un fichier index.js.

Il va comprendre une variable router qui va invoquer la méthode Router du framework Express, elle va être utilisée pour définir des routes dans notre application.
